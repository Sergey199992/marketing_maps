# Промо-карта для маркетинга 🗺️

**Версия:** `v.YS2026_06_19_marketing_v15`  
Веб-приложение для контроля расклейки рекламных материалов, визуализации геоданных и автоматического формирования отчётности. 

Приложение автоматически размещает на карте точки на основе фотографий с GPS-координатами, текстовых адресов или координат из файлов. Работает непосредственно в любом браузере и не требует установки.

---

### 👥 Для кого разработано
* **Маркетологи** — для оценки охвата и планирования кампаний.
* **Бригады расклейки** — для быстрой фиксации выполненных работ.
* **Руководители проектов наружной рекламы** — для оперативного контроля подрядчиков.

---

### 🎯 Какие задачи решает
* **Контроль подрядчиков:** Проверка факта расклейки по реальным фотоотчётам от бригад.
* **Автоматизация отчётности:** Мгновенное формирование готовых отчетов в Excel (CSV).
* **Наглядная визуализация:** Отображение всех точек мониторинга на единой интерактивной карте.
* **Цветовая классификация:** Разделение точек по статусам (например: *сделано* / *не сделано* / *конкуренты*).
* **Командная работа:** Быстрый обмен файлами проектов между сотрудниками.
* **Аналитика покрытия:** Поиск «слепых зон» и оценка плотности распределения рекламы.

### 📖 Подробное описание ключевых функций

#### 📷 Автоматическая загрузка фото
Основной и самый быстрый способ построения карты. 
* Поддерживаются форматы `JPG` / `JPEG`.
* Возможна массовая загрузка (сотнями файлов за один раз) с отображением прогресс-бара.
* **Двойная защита данных:** координаты автоматически извлекаются из EXIF-метаданных фотографии, либо парсятся из названия файла, если метаданные были стёрты мессенджерами.

#### ✋ Ручное добавление точек
Позволяет быстро нанести новые объекты на карту без файлов.
* Нажмите кнопку **✋ Вручную**, после чего кликайте мышью в нужных местах карты.
* Для выхода из режима редактирования нажмите кнопку повторно или используйте клавишу `ESC`.

#### 🎯 Выделение и раскраска групп
Инструмент для быстрого менеджмента и категоризации сотен точек.
* Нажмите кнопку **🎯 Выделить**, выберите нужный цвет из палитры и нарисуйте прямоугольник вокруг группы маркеров.
* Все точки, попавшие в область, мгновенно изменят свой цвет.

#### 🌈 Интеграция с Яндекс.Картами (Цветной KML)
Основной формат для обмена данными с *Яндекс Конструктором карт*.
* **Импорт:** Приложение считывает цвета точек, настроенные в Яндексе, и сохраняет их в интерфейсе.
* **Экспорт:** Сформированный файл полностью совместим с Яндексом — цвета ваших маркеров корректно отобразятся в Конструкторе карт.

#### 🌐 Одноцветный KML
Стандартный формат геоданных без сохранения цветовой палитры. Используется для обеспечения максимальной совместимости с другими картографическими и геоинформационными системами (GIS).

#### 🔢 TXT координаты
Обмен данными в виде простого текстового списка. Каждая строка файла должна содержать структуру: `широта,долгота`. Удобно для интеграции со сторонними GPS-трекерами подрядчиков.

#### 📍 TXT адреса
Импорт текстового списка адресов (по одному адресу на строку). Приложение имеет встроенный инструмент геокодирования: адреса автоматически преобразуются в точные географические координаты на карте со скоростью около 1 секунды на один адрес.

#### 📋 Главный отчёт (CSV)
Итоговый структурированный файл для детального анализа в Microsoft Excel или Google Таблицах. Содержит следующие колонки:
1. `Район`
2. `Адрес`
3. `Широта`
4. `Долгота`
5. `Фото`
6. `Вид рекламы`
7. `Дата`
8. `Комментарий`

#### 💾 Файл проекта (JSON)
Позволяет полностью сохранить текущую рабочую сессию. Внутри файла резервируются все добавленные точки, привязанные фотографии, кастомные настройки интерфейса и цветовые маркеры. Вы можете переслать этот файл коллеге для продолжения работы.

#### 🔍 Умный поиск
Строка поиска в шапке приложения является универсальной. Она принимает:
* Названия городов и населенных пунктов.
* Конкретные улицы и номера домов.
* Прямые географические координаты.
Для перехода к объекту введите запрос и нажмите `Enter`.

#### 🗺️ Картографические подложки
Для удобства ориентирования в разных типах местности в приложение интегрировано 5 вариантов карт:
1. **OpenStreetMap** (схематическая народная карта)
2. **2GIS** (детальная городская карта с организациями)
3. **Google схема** (классическая карта дорог)
4. **Google спутник** (высокодетализированные снимки из космоса)
5. **Google гибрид** (спутниковые снимки с наложением названий улиц и дорог)

#### 🔧 Кастомизация маркеров
Специальный ползунок в шапке приложения позволяет плавно регулировать размер точек на карте. Это помогает комфортно работать как с редкими точками на уровне страны, так и с огромной плотностью расклейки в рамках одного жилого квартала.

### 📱 Инструкция для расклейщиков: Как правильно фотографировать

Для корректной работы приложения рекомендуется использовать бесплатное мобильное приложение **Conota Camera** (доступно на Android и iOS). 

В настройках Conota Camera **обязательно включите два параметра**:
*   **✅ Координаты на фото (Водяной знак):** Геоданные пропечатываются прямо на самом изображении.
*   **✅ Координаты в имени файла:** Название генерируемого файла автоматически принимает вид `57.33402,61.92401_фото.jpg`.

> **💡 Почему это важно?** Это обеспечивает **двойную защиту данных**. Если при отправке отчёта мессенджеры или сбои сети повредят внутренние GPS-метаданные (EXIF) снимка, алгоритм Промо-карты автоматически извлечет точные координаты из названия файла.

### ⚠️ Правила передачи фотографий (Без потери GPS)

При обычной отправке картинок большинство мессенджеров сжимают их и **полностью удаляют геоданные**. Чтобы этого избежать, используйте следующие каналы передачи:

#### 🥇 Надёжные способы (Метаданные сохраняются на 100%):
*   **Облачные хранилища:** Загрузка папки с фото на *Яндекс.Диск*, *Google Диск* или *Dropbox* с последующей отправкой ссылки.
*   **Прямая передача:** Копирование файлов через *USB-кабель*, *AirDrop* или *Bluetooth*.
*   **Электронная почта:** Отправка снимков по *Email* в качестве прикрепленных файлов (вложений).

#### 🥈 Передача через мессенджеры (Строго как файлы!):
Если отчёт отправляется в чат, пересылайте снимки **НЕ из галереи**, а как документы:
*   **Telegram:** Нажмите `📎 Иконку скрепки` → выберите пункт **Файл** (без сжатия).
*   **WhatsApp:** Нажмите `📎 Иконку скрепки` → выберите пункт **Документ**.

### 📊 Источники геоданных (Отображение в статистике)

Приложение автоматически анализирует каждый загруженный объект и категоризирует его в модуле статистики по источнику происхождения:

*   **`EXIF`** — Координаты успешно считаны напрямую из внутренних метаданных фотографии (максимальная точность).
*   **`Название`** — Данные EXIF отсутствовали, но координаты были успешно восстановлены из имени файла.
*   **`Адреса`** — Точки, полученные в результате текстового импорта списка адресов и последующего геокодирования.
*   **`Вручную`** — Маркеры, поставленные кликом по карте, либо импортированные из файлов KML / TXT.
*   **`Без GPS`** — Проблемные фотографии без каких-либо геоданных. Они **не отображаются на карте** и требуют проверки.

### 🎯 Типовые сценарии использования (Use Cases)

1.  **🕵️ Глубокий анализ конкурентов**  
    Экспортируйте карту активности конкурентов из *Яндекс Конструктора* в формате KML и загрузите в приложение. Вы наглядно увидите зоны присутствия оппонентов и сможете точно определить свободные районы для вашей расклейки.
2.  **📊 Сверка адресной базы**  
    Выгрузите из вашей CRM или Excel-таблицы список целевых адресов ➡️ импортируйте TXT-файл в приложение ➡️ визуально оцените плотность покрытия на карте ➡️ при необходимости скорректируйте данные и сделайте обратный экспорт в CSV для отчетности.
3.  **🔧 Восстановление «битых» фотографий**  
    Если куратор прислал снимки, которые полностью потеряли GPS-теги, вам не нужно заставлять его переделывать работу. Достаточно узнать координаты места, переименовать файл в формат `57.33402,61.92401_комментарий.jpg` и загрузить в систему — точка встанет идеально.
4.  **⚡ Молниеносная повторная выгрузка**  
    Все импортированные адреса кэшируются внутри сессии приложения. Если вам нужно повторно выгрузить главный CSV-отчёт для тех же точек, процесс генерации файла займет доли секунды.
5.  **📸 Сверхмассовая обработка данных**  
    Приложение оптимизировано под высокие нагрузки. Пакет из **500+ фотографий** обрабатывается интеллектуально — небольшими порциями, не перегружая оперативную память браузера, с выводом наглядного прогресс-бара.
