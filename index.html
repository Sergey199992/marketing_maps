📘 СПРАВКА · Промо-карта

Версия: v.YS2026_06_19_marketing_v15

Промо-карта — веб-приложение для контроля расклейки рекламных материалов и формирования отчётности.

Автоматически размещает на карте точки по фотографиям с GPS-координатами, адресам или координатам из файлов. Работает в любом браузере без установки.

Для кого: маркетологи, бригады расклейки, руководители проектов наружной рекламы.

🎯 Какие задачи решает

  • Контроль факта расклейки по фотоотчёту от бригад
  • Автоматическое формирование отчёта в Excel (CSV)
  • Визуализация всех точек на одной карте
  • Классификация точек цветом (сделано / не сделано / конкуренты)
  • Обмен проектами между сотрудниками
  • Анализ покрытия и зон без расклейки

📋 Функции приложения

  • Загрузка фото с GPS
  • Ручное добавление точек
  • Выделение и раскраска групп
  • Импорт цветного KML (Яндекс)
  • Экспорт цветного KML (Яндекс)
  • Импорт одноцветного KML
  • Экспорт одноцветного KML
  • Импорт TXT (координаты)
  • Импорт TXT (адреса)
  • Экспорт TXT (координаты)
  • Экспорт TXT (адреса)
  • Экспорт CSV (главный отчёт)
  • Сохранение/открытие проекта
  • Поиск городов и адресов
  • 5 подложек карт
  • Настройка размера точек

📖 Описание функций

📷 Загрузка фото — основной способ добавления точек. Поддерживаются JPG/JPEG, загружаются сотнями файлов за раз с прогресс-баром. Координаты берутся из метаданных или из имени файла (двойная защита).

✋ Ручное добавление точек — нажмите кнопку ✋ Вручную, затем кликайте по карте в нужных местах. Для выхода — повторное нажатие или клавиша ESC.

🎯 Выделение и раскраска — нажмите 🎯 Выделить, выберите цвет из палитры и нарисуйте прямоугольник мышью вокруг группы точек. Все точки внутри автоматически перекрасятся.

🌈 KML Яндекс (цветной) — основной формат обмена с Яндекс Конструктором карт. При импорте сохраняются цвета точек из Яндекса, при экспорте — ваши цвета корректно отображаются в Яндекс Конструкторе.

🌐 KML одноцветный — стандартный формат KML без сохранения цветов. Используется для совместимости с другими системами.

🔢 TXT координаты — простой список, каждая строка содержит широта,долгота. Подходит для обмена с подрядчиками.

📍 TXT адреса — список адресов, по одному на строку. При импорте адреса автоматически геокодируются в координаты (≈1 сек на адрес).

📋 CSV (главный отчёт) — итоговый файл для Excel с колонками: Район, Адрес, Широта, Долгота, Фото, Вид рекламы, Дата, Комментарий.

💾 Проект (JSON) — полное сохранение всех данных: точки, фото, настройки, цвета.

🔍 Поиск — поле поиска принимает название города, улицу или координаты. Нажмите Enter.

🗺 Подложки карт — 5 вариантов: OpenStreetMap, 2GIS, Google схема, Google спутник, Google гибрид.

🔧 Размер точек — ползунок в шапке меняет размер маркеров.

📱 Как правильно фотографировать

Используйте приложение Conota Camera (бесплатно, Android/iOS). В настройках обязательно включите:

  • ✅ Координаты на фото (Водяной знак) — координаты пропечатываются прямо на снимке
  • ✅ Координаты в имени файла — название файла становится вида 57.33402,61.92401_фото.jpg

Почему это важно: это двойная защита. Если GPS-метаданные потеряются при передаче — координаты автоматически возьмутся из имени файла.

⚠️ Как передавать фото без потери GPS

🥇 Надёжные способы (метаданные гарантированно сохраняются):

  • Облачные хранилища: Яндекс.Диск, Google Диск, Dropbox
  • Прямая передача: USB-кабель, AirDrop, Bluetooth
  • Email: отправляйте фото как вложения

🥈 Через мессенджеры — только как файлы, НЕ как фото из галереи:

  • Telegram: 📎 → Файл (НЕ Галерея!)
  • WhatsApp: 📎 → Документ

При отправке фото из галереи мессенджеры удаляют метаданные — координаты будут потеряны.

📊 Источники точек (видно в статистике)

  • ✅ EXIF — координаты из метаданных фото (самый надёжный)
  • 📝 Название — восстановленные из имени файла
  • 📍 Адреса — из импорта TXT-адресов
  • ✋ Вручную — клик по карте, KML/TXT импорт
  • ❌ Без GPS — фото без координат (не отображаются)

🎯 Типовые сценарии использования

🕵️ Анализ конкурентов. Загрузите KML конкурентов из Яндекс Конструктора — увидите где они работают и где есть свободные зоны.

📊 Сверка с базой. Выгрузите из Excel адреса → импорт TXT → карта → экспорт CSV обратно → сверка.

🔧 Восстановление битых фото. Если фото потеряли GPS — переименуйте в 57.33402,61.92401_фото.jpg и загрузите заново.

⚡ Быстрая повторная выгрузка. Адреса кэшируются — вторая выгрузка CSV тех же точек происходит мгновенно.

📸 Массовая загрузка. 500+ фото за раз обрабатываются порциями с прогресс-баром.

🔧 Горячие клавиши

  • ESC — выйти из ручного режима / режима выделения / закрыть окно
  • Enter в поле поиска — найти адрес или координаты
